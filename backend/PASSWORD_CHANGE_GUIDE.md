### 2. 네이버 메일 앱 비밀번호
**현재 (노출됨)**: `2GFH6M3WHJS1`

**새 앱 비밀번호 발급 방법**:
1. 네이버 메일 로그인
2. 설정 → POP3/IMAP 설정
3. 기존 앱 비밀번호 삭제
4. 새 앱 비밀번호 생성
5. 16자리 새 비밀번호 복사

### 3. 강력한 비밀번호 생성 규칙
- **최소 16자 이상**
- **대문자, 소문자, 숫자, 특수문자 포함**
- **예시**: `Prepmood2025!@#$MySQL`
